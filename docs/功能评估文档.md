## 一、架构概览

### 1.1 技术栈

| 层级 | 技术选型 | 版本 |
|------|---------|------|
| 跨端框架 | Taro | 4.1.1 |
| UI 框架 | React | 18.2.0 |
| 移动端 | React Native | 0.73.11 |
| 容器 | Expo | SDK 50 |
| 状态管理 | Zustand | 4.5.2 |
| 样式 | SCSS → RN StyleSheet | 自定义 Metro Transformer |
| 语言 | TypeScript | 5.4.5 |

### 1.2 项目结构

```
src/
├── app.tsx              # 应用入口（Taro App 生命周期）
├── app.config.ts        # 应用配置（路由、TabBar、窗口）
├── app.scss             # 全局样式（⚠️ RN 下为空，见问题 #1）
├── components/          # 公共组件（5 个）
├── pages/               # 页面（47 个，按功能域划分）
├── services/            # API 服务层（12 个 service）
├── stores/              # 状态管理（6 个 Zustand store）
├── types/               # TypeScript 类型定义（5 个文件）
├── platform/            # 平台适配层（RN / 小程序 / H5）
└── utils/               # 工具函数（5 个文件）
```

### 1.3 底部导航结构

| Tab | 图标 | 页面路径 | 关联页面数 |
|-----|------|---------|-----------|
| 行程圈 | home | pages/index/index | 12 个 |
| 发现 | discover | pages/discover/index | 2 个 |
| 发布 | create | pages/trip-create/index | 2 个 |
| 消息 | message | pages/messages/index | 2 个 |
| 我的 | profile | pages/profile/index | 29 个 |
