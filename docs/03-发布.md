### 数据流

```
页面加载
    → 检查 ?template= 参数
        → templateService.getTemplateDetail(id) → 预填表单
    → 检查本地草稿 → loadDraft() → 预填表单
        ↓
表单填写（自动保存草稿 → saveDraft()）
    → 校验
        → validateTripTitle(title)      // 必填 + 2-50 字符
        → validateTripDescription(desc)  // 最长 1000
        → 时间必填检查
            ↓
    useTripStore.createTrip(dto)
        → tripService.createTrip(dto)
            → POST /trips
        ↓
    乐观更新 trips/myTrips 列表
        ↓
    清除草稿 → clearDraft()
        ↓
    跳转行程详情页
```

### 依赖关系

- **Store**：`useTripStore`（createTrip, isLoading）
- **Service**：`tripService.createTrip`, `templateService.getTemplateDetail`, `uploadService.uploadImage`
- **工具**：`validateTripTitle`, `validateTripDescription`
- **常量**：`TRIP_TYPES`, `COST_TYPES`, `VISIBILITY_OPTIONS`, `JOIN_RULES`, `MAX_TITLE_LENGTH`, `MAX_DESCRIPTION_LENGTH`
- **平台**：`platformService.showToast`, `platformService.redirectTo`, `platformService.chooseImage`, `platformService.getLocation`, `platformService.showLoading`, `platformService.hideLoading`
