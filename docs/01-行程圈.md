| 功能 | 状态 | 实现细节 |
|------|------|---------|
| 行程列表 | ✅ | `TripCard` 组件，每行一个行程卡片 |
| 分页加载 | ✅ | `PAGE_SIZES.TRIP_LIST = 10`，上拉触发 `fetchTrips(false)` |
| 下拉刷新 | ✅ | `fetchTrips(true)` 重置 page=1 |
| 空状态 | ✅ | `EmptyState` 组件，图标 🌟，引导「发起行程」 |

| FAB 按钮 | ✅ | 右下角 + 号，跳转 `trip-create` |
| 分享 | ❌ | `useShareAppMessage` 已移除（RN 不支持） |

### 数据流

```
useLoad → useTripStore.fetchTrips(true)
                ↓
         tripService.getTrips(page, 10)
                ↓
         GET /trips?page=1&pageSize=10
                ↓
         set({ trips: response.list, hasMore: list.length === 10 })
```

### 依赖关系

- **Store**：`useTripStore`（trips, isLoading, isLoadingMore, hasMore, fetchTrips）
- **Service**：`tripService.getTrips`
- **组件**：`TripCard`, `EmptyState`, `LoadingView`
- **平台**：`platformService.navigateTo`

| # | 严重度 | 问题 |

| 1 | 🟡 | 无搜索/筛选功能，无法按类型/地点过滤 |

### 数据流

```
useLoad → router.params.id
    → useTripStore.fetchTripDetail(tripId)
        → tripService.getTripDetail(tripId)
            → GET /trips/:id
    → commentService.getComments(tripId)  [评论 Tab]

报名 → useTripStore.joinTrip(tripId, status)
    → tripService.joinTrip(tripId, status)
        → POST /trips/:id/join
    → 乐观更新 trips/myTrips/currentTrip
```

### 权限控制

```typescript
canEditTrip(user, trip)          // 创建者才能编辑
canManageParticipants(user, trip) // 创建者才能管理参与者
canJoinTrip(user, trip)          // 非创建者 + 未参加 + 已发布 + 未满
canComment(user, trip)           // 参加者或创建者
canShareTrip(user, trip)         // 非 private
```
